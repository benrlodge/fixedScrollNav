About fixedScrollNav.js

fixedScrollNav is a simple lightweight plugin that provides the framework for a (position) fixed navigation menu whose links toggle active states based on the section it is on.

How to Use

Build the page

<!-- THE NAVIGATION -->
<nav>
<ul>
<li> <a class='nav-link' href="#article1"> Link 1 </a> </li>
<li> <a class='nav-link' href="#article2"> Link 2 </a> </li>
<li> <a class='nav-link' href="#article3"> Link 3 </a> </li>
</ul>
</nav>

<!-- THE DESTINATION -->
<section>
<article id="#article1">
...
</article>

<article id="#article2">
...
</article>

<article id="#article3">
...
</article>
</section>

Load up the scripts

<script src="jquery.js"> </script>
<script src="fixedScrollNav.js"> </script>

Power up the plugin

$('body').fixedScrollNav()

All Options

settings = sections : 'article' # Sections linkType : 'individual' # Choose individual or group links : '.nav-link' # Navigation Links activeClass : 'active' # Navigation link toggle class animated : true scrollSpeed : 900 threshold : ''
