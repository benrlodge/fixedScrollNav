~ About fixedScrollNav.js ~

fixedScrollNav is a simple lightweight plugin that provides the framework for a (position) fixed navigation menu whose links toggle active states based on the section it is on.

~ How to Use ~

1. Build the page

First create an id with a unique name for each section you want associated with the navigation.

<!-- THE DESTINATION -->
<section>
<article id="#article1">
...
</article>

<article id="#article2">
...
</article>

<article id="#article3">
...
</article>
</section>

Next, build your navigation menu. Set the href value equal to hash + the id you created with the associated section.

<!-- THE NAVIGATION (with no extra options) -->
<nav>
<ul>
<li> <a href="#article1"> Link 1 </a> </li>
<li> <a href="#article2"> Link 2 </a> </li>
<li> <a href="#article3"> Link 3 </a> </li>
</ul>
</nav>

<!-- or: THE NAVIGATION with class name (details in options below) -->
<nav>
<ul>
<li> <a class='nav-link' href="#article1"> Link 1 </a> </li>
<li> <a class='nav-link' href="#article2"> Link 2 </a> </li>
<li> <a class='nav-link' href="#article3"> Link 3 </a> </li>
</ul>
</nav>

Load up the scripts

<script src="jquery.js"> </script>
<script src="fixedScrollNav.js"> </script>

Power up the plugin

$("body").fixedScrollNav(
nav : "#myNav"
)

~ All the Options ~

sections : ".mysections"
group : false
links : ".nav-link"
nav : "#mynav"
activeClass : "myActiveClass"
scrollSpeed : 900
scroll : false
threshold : 100

sections

Give the "sections" option a value that you want associated with each section. This tells the scrolling function when to toggle the active class in the navigation menu.

group, links, & nav

By default, the plugin will assume each link within your navigation menu (set with the "nav" option) has a corresponding section. If you would like to specify which links are associated with the sections (for example, if your navigation also contains external links), set linkType to "individual" and give each internal scrolling link a classname, and include it with option "links"

nav

If you want to use the default "grouping" navigation link method, tell the plugin where your links live with the "nav" option.

activeClass

By default, the plugin will toggle a class named "active" with the menu links as the viewer scrolls through the page, changing sections. If you would like to use your own custom class name, use the "activeClass" option.
Please note: you do not need to include a " . " with the class name. For example do not do -> nav: ".myActiveClassName", instead, do -> "myActiveClassName"

scrollSpeed

The plugin includes a smooth scroll to each section when a navigation link is clicked. The speed is set to 900 milliseconds by dafault. If you would like to alter the speed, do so using the "scrollSpeed" option

scroll

If you do not want an animated scroll to each section, set "scroll" to false

threshold

By default, the plugin will toggle the active class in the navigation menu with the associated section shown on the screen as soon as the section is 1px from the top of the viewport. If you would like to change this threshold, set the number of pixels using the "threshold" option. For example, if you would like the active class to toggle when the section is 100px from the top of the viewport, set threshold to 100. (This website used 150px as its threshold) Please keep in mind: The plugin will include padding as part of the section.
