~ About fixedScrollNav.js

fixedScrollNav.js is a simple lightweight plugin that provides the framework for a (position) fixed navigation menu whose links toggle active states based on the section of the page that is in the viewport. Scroll down the page to see it in action

~ How to Use

1. Build the page

First create an id with a unique name for each section you want associated with the navigation.

<!-- THE DESTINATION -->

<section>
<article id="#article1">
...
</article>

<article id="#article2">
...
</article>

<article id="#article3">
...
</article>
</section>

Next, build your navigation menu. Set the href value equal to hash + the id you created with the associated section.

<!-- THE NAVIGATION (default) -->

<nav>
<ul id="myNav">
<li> <a class="nav-link" href="#article1"> Link 1 </a> </li>
<li> <a class="nav-link" href="#article2"> Link 2 </a> </li>
<li> <a class="nav-link" href="#article3"> Link 3 </a> </li>
</ul>
</nav>



<!-- THE NAVIGATION (with class names - see options below) -->

<nav>
<ul id="myNav">
<li> <a class="nav-link" href="#article1"> Link 1 </a> </li>
<li> <a class="nav-link" href="#article2"> Link 2 </a> </li>
<li> <a class="nav-link" href="#article3"> Link 3 </a> </li>
</ul>
</nav>

2. Load up the scripts

<script src="jquery.js"> </script>
<script src="fixedScrollNav.js"> </script>

3. Power up the plugin

$("body").fixedScrollNav(
nav : "#myNav"
)

~ All the Options

sections : ".mysections"
nav : "#myNav"
defaultNav : false
links : ".nav-link"
activeClass : "myActiveClass"
scrollSpeed : 900
threshold : 100

sections

The sections option sets which tags or classes divide your page into the sections that will be associated with the internal links in the navigation. The default option is article tags.

nav, defaultNav & links

By default, the plugin will assume each link within your navigation menu (set with the "nav" option) has a corresponding section on the page. If you would like to specify which links in the navigation are associated with the sections on the page, set the defaultNav option to true, and give each link you would like to associate with a section a classname, and include it in the "links" option. See HTML example above, and JavaScript example below.

Example:

$("body").fixedScrollNav({
defaultNav : true,
nav : "#myNav",
links : ".nav-link"
})

activeClass

By default, the plugin will toggle a class named "active" with the navigation menu links as the viewer scrolls through the page. If you would like to use your own custom class name, set it using the "activeClass" option.
Please note: you do not need to include a " . " with the class name.
For example, do not do nav: ".myActiveClassName"
instead, do nav: "myActiveClassName"

scrollSpeed

The plugin includes a smooth scroll to each section when a navigation link is clicked. The speed is set to 900 milliseconds by dafault. If you would like to alter the speed, do so using the "scrollSpeed" option. Set speed to 0 to disable animation.

threshold

By default, the plugin will toggle the active class in the navigation menu with the associated section shown on the screen as soon as the section is 1px from the top of the viewport. If you would like to change this threshold, set the number of pixels using the "threshold" option. For example, if you would like the active class to toggle when the section is 100px from the top of the viewport, set threshold to 100. Please keep in mind: The plugin will include padding as part of the section.
